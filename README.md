# Daraz Automation using Selenium 

## Using 
* python 
* Selenium 
* BeautifulSoup

## How to use 

- `git clone https://github.com/adilshehzad786/Selenium-Automation.git`
- python -m venv venv
- cd venc/Scripts/activate
- pip install -r requirements.txt
- python main.py

### For Python 3.9 

`py main.py`
